# Trae Neo Themes

This package contains the Trae-inspired VS Code color themes and product icon theme.

## Included
- `Trae Neo Dark`
- `Trae Neo Light`
- `Trae Neo Product Icons`
- `fonts/trae-codicon.ttf`

## Notes
Choose `Trae Neo Dark` or `Trae Neo Light` as the color theme, and choose `Trae Neo Product Icons` as the product icon theme inside VS Code.
